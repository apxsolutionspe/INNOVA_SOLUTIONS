# IA Analytics

## Descripcion

Analisis gerencial preparado para reglas internas o proveedor externo de IA.

## Variables

- `AI_MODE`
- `AI_PROVIDER`
- `AI_API_KEY`
- `AI_MODEL`

## Endpoints internos

- `GET /api/ai-analytics/business-summary`
- `GET /api/ai-analytics/sales-insights`
- `GET /api/ai-analytics/inventory-insights`
- `GET /api/ai-analytics/profitability-insights`
- `POST /api/ai-analytics/ask`
- `POST /api/ai-analytics/test-connection`

## Seguridad

Antes de usar IA real se deben resumir datos y excluir passwords, tokens, documentos privados o informacion innecesaria.
